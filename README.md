# Personalized Financial AI Advisor

A hackathon project that provides **personalized financial guidance**
based on user income, expenses, savings, goals, and risk tolerance.

##  Features
- User-specific financial advice
- Monthly savings calculation
- Risk-based investment suggestions
- Interactive dashboard (Streamlit)
- AI-powered logic (mocked / API-based)

##  How It Works
1. User enters financial details
2. Data is processed using Python
3. Advisor generates tailored insights
4. Results are shown with charts and explanations

##  Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- OpenAI (optional)

```bash
pip install -r requirements.txt
